# Glitch Engine

An AI-powered game engine — a custom fork of Godot 4.6.3-stable.

Glitch Engine extends Godot with integrated AI systems, world-building tools,
NPC generation, quest building, asset management, AI debugging assistance,
and AI-assisted game creation workflows.

## Current Status

Phase 1 — Engine Foundation. See [PROJECT_STATUS.md](PROJECT_STATUS.md).

## Godot Source Setup

The Godot source code is not stored in this repository (file size).
After cloning this repo, run the following to pull the Godot source locally:

```bash
git clone --branch 4.6.3-stable --depth 1 https://github.com/godotengine/godot.git godot-source
```
