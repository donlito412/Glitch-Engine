NEXT STEP — Install the plugin in Glitch Engine:

1. Open Glitch Engine editor
2. In the editor menu: Project → Project Settings → Plugins
3. Find 'GlitchAI Assistant' and click Enable
4. A 'GlitchAI' tab will appear at the bottom of the editor
5. Click Settings → enter your Anthropic API key → Save Key
6. Type any message to start chatting with GlitchAI

NOTE: The plugin files go in your GAME PROJECT's addons/ folder.
For testing, create a new project in Glitch Engine and copy
the plugin folder there, OR we set up a test project in Phase 3.
